Publica tu App FumiMat gratis en la Nube con GitHub Pages

Sigue estos pasos rápidos para poder acceder a la herramienta desde tu celular iOS o Android.

Paso 1: Preparar tu repositorio

  1. Crea una cuenta en GitHub.
  2. Crea un nuevo repositorio público llamado fumimat.
  3. Haz clic en "Create new file" para crear un archivo en el repositorio.
  4. Nómbralo exactamente como index.html.
  5. Pega todo el código de este archivo dentro y haz clic en "Commit changes...".

Paso 2: Activar el hosting gratuito

  1. Dentro de tu repositorio en GitHub, ve a la pestaña Settings (Configuración) arriba.
  2. En la barra lateral izquierda, haz clic en la sección Pages.
  3. Bajo la opción Build and deployment, selecciona la rama main (o master) y el directorio / (root).
  4. Presiona Save (Guardar).
  5. ¡Listo! En unos minutos obtendrás tu dirección web pública para guardarla como ícono en la pantalla de inicio de tu smartphone.
